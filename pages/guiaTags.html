Guia de Tags HTML

Este site explica as tags de HTML

< !DOCTYPE html> - Declaração do Documento

Informa ao navegador que este arquivo é um documento HTML

< head> - Cabeçalho

Contém informações sobre a página que não aparecem para o usuário: título da página, links de estilo, meta tags...

< title> - Título

Define um título que aparece na aba do navegador

Exemplo: esta página tem o título: "Guia de Tags HTML"

< body> - Corpo da página

É onde fica todo o conteúdo visível da página: Textos, Imagens, links, tabelas...

< h1> até < h6> - Títulos

As tags de título vão de < h1> (mais importante) até < h6> (menos importante).

< b> deixa o texto em negrito. Já < strong> também deixa em negrito, mas indica que o conteúdo é importante.

< i> deixa o texto em itálico

< ul > e < ol > - Listas

< ul > cria listas não ordenadas (com marcadores).
< ol > cria listas ordenadas (com números).

Exemplo de lista não ordenada

  • Teste 1
  • Teste 2
  • Teste 3

Exemplo de lista ordenada

  1. Teste 1
  2. Teste 2
  3. Teste 3

< table > - Tabelas

Exibe dados em formato de linhas e colunas.
Contém uma estrutura de 3 tags para formar a tabela, table, tr e th.
Table: determina o início da tabela.
tr: determina uma linha.
th: determina uma coluna.
